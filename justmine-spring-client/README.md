# justmine suite配套管理平台
## 拉取infra配置
1. 业务应用拉取基础配置
2. 基础配置版本化，支持回滚
3. 业务应用可以固定基础配置版本
4. 业务应用可以自定义覆盖基础配置
5. 基础应用配置为业务应用的默认配置，配置优先级别上，业务应用配置高于基础配置
